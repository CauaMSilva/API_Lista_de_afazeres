## API REST com Node.js, Express, Sequelize e SQLite

Esse projeto é uma **API RESTful** desenvolvida com **Node.js** e **Express**, utilizando o **Sequelize ORM** para gerenciar o banco de dados **SQLite**.  
O objetivo é demonstrar o uso de um ORM para criar, ler, atualizar e deletar registros (CRUD) de forma prática e organizada.

---

## Tecnologias Utilizadas

- **Node.js**
- **Express**
- **Sequelize (ORM)**
- **SQLite**
- **Nodemon** (para desenvolvimento)

---

## Funcionalidades

- Criação de registros
- Leitura de dados
- Atualização de informações
- Exclusão de registros
- Integração com ORM Sequelize
- Banco de dados SQLite

---

## Conceitos Aplicados

- CRUD (Create, Read, Update, Delete)
- Arquitetura MVC
- Conexão e modelagem de dados com ORM
- Boas práticas de API RESTful
- Manipulação de banco de dados relacional com SQLite
